# Character model credits

Player character 3D models (glTF/GLB), all freely usable:

- **RobotExpressive.glb** — by Tomás Laulhé, modified by Don McCurdy. License: CC0
  (public domain). From the three.js examples.
- **Xbot.glb** — Adobe Mixamo character ("X Bot"), distributed with the three.js
  examples. Free to use.
- **Soldier.glb** — Adobe Mixamo character, distributed with the three.js examples.
  Free to use.

Source: https://github.com/mrdoob/three.js/tree/master/examples/models/gltf

The default "Sonic" model remains embedded in EndlessRunner.js (unchanged).
