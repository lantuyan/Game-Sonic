# Character model credits

Selectable 3D character models (glTF/GLB), all freely usable:

- **RobotExpressive.glb** — by Tomás Laulhé, modified by Don McCurdy. License: CC0
  (public domain).
- **Horse.glb**, **Parrot.glb** — classic three.js animated models (mirada / "ro.me"
  WebGL Animation assets), distributed with the three.js examples and free to use.

Source: https://github.com/mrdoob/three.js/tree/master/examples/models/gltf

The default "Sonic" model remains embedded in EndlessRunner.js (unchanged).
